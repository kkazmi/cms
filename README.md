# 🛠️ Complaint Management System

## 📌 Overview
The **Complaint Management System (CMS)** is a web-based application designed to manage, track, and resolve complaints efficiently. It allows users to register complaints, while Admins and staff can monitor, assign, and resolve them in a structured workflow.

---

## 🎯 Features

### 👤 User Panel
- Register/Login
- Submit new complaint
- Attach documents/images
- Track complaint status
- View complaint history

### 🧑‍💼 Admin Panel
- Dashboard with analytics
- Manage users
- View all complaints
- Assign complaints to staff
- Update complaint status
- Generate reports

### 👨‍🔧 Staff/Agent Panel
- View assigned complaints
- Update progress
- Resolve complaints
- Add remarks/comments

---

## 🔄 Complaint Workflow
1. User submits complaint  
2. Complaint is registered with unique ID  
3. Admin assigns complaint to staff  
4. Staff works on issue  
5. Status updated (Pending → In Progress → Resolved)  
6. User gets notified
